# 🎓 Student Placement Prediction using Logistic Regression

This project is a **Machine Learning Web App** that predicts whether a student will be placed or not using **Logistic Regression**.
The model is deployed using **Streamlit** and trained on student placement dataset.

---

## 🚀 Features

* Predict student placement
* Logistic Regression model
* Streamlit Web App
* Clean UI input form
* Real-time prediction

## ▶️ Run Locally

Step 1 — Clone repo

```
git clone https://github.com/kaushalsharma02/student-placement-prediction.git
```

Step 2 — Go to folder

```
cd student-placement-prediction
```

Step 3 — Install requirements

```
pip install -r requirements.txt
```

Step 4 — Run Streamlit

```
streamlit run app.py
```
